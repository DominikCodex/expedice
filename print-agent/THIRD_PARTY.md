# Third-party components

## SumatraPDF

Print agent uses SumatraPDF for silent PDF printing on Windows.

- Website: https://www.sumatrapdfreader.org/
- Download page: https://www.sumatrapdfreader.org/download-free-pdf-viewer
- Version prepared by scripts: 3.6.1 64-bit portable
- License: GPLv3

The install script downloads the official portable ZIP when `bin\SumatraPDF.exe` is not bundled locally.
